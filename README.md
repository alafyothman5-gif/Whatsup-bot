# WhatsPriceBot V4.2 Clean Admin

نسخة منظمة لإصلاح لوحة الأدمن:

- لا يوجد Auto Refresh يطفي ويفتح الصفحة.
- إضافة المنتجات تعمل من اللوحة.
- تعديل المنتجات يعمل من اللوحة.
- خانات سعر الشريط، سعر العلبة، عدد الشرائط.
- صندوق وارد للأدمن للرد على الزبائن.
- قراءة صور المنتجات مع تحويل أي شك للأدمن.
- الروشتات والأسئلة الطبية تتحول للأدمن.

Admin:

```text
/admin?token=admin2026
```

لا ترفع ملف `.env` إلى GitHub.
